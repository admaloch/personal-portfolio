Photo Order Transfer Tool

Windows batch script designed to streamline photo orders

Github
Project details:

This tool is specifically built to assist in locating files for invoice orders, copying them over, keeping track of files that have not yet been uploaded, and maintaining accurate records for each processed order. Previously, the staff had to manually check if the photos requested were located on the cloud and copy them to a new location. This process was time-consuming, especially considering the large size of the directory and the high-resolution prints that took a long time to load and copy. The primary goal of building this tool was to streamline the entire process, allowing the staff to focus on other projects while the search is being completed. This tool automates the file retrieval and copying process, saving time and effort for the staff.
